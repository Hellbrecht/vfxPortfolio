# JP Marchand Portfolio

A dynamic React portfolio built with Vite for a VFX Environment Supervisor and Artist.

## Run Locally

```bash
npm install
npm run dev
```

The local dev server runs at `http://127.0.0.1:5173/` by default.

## Customize

Edit `src/portfolioData.js` to update biography, IMDb credit tiles, poster URLs, roles, experience, showreel links, and contact details.

The showreel is embedded from Vimeo so the deployed site can stay lightweight and GitHub Pages-friendly.

## Build

```bash
npm run build
```
